# Spotify Analytics Dashboard
![Spotify Dashboard](اسم_الصورة.png)
Interactive music analytics dashboard built with Power BI to analyze song popularity, artist performance, and album trends through dynamic data visualizations.

## Tools Used
- Power BI
- Excel
- Data Visualization

## Dashboard Features
- Song popularity analysis
- Artist performance insights
- Album type comparison
- Monthly trends visualization

## Project Goal
Transform music data into visual insights to understand listening trends and artist performance.
